#### About

Open Fodder Editor is a campaign and map editor for [Open Fodder](https://github.com/OpenFodder/openfodder) and Cannon Fodder

#### To Compile

* SDL2
* SDL2-Mixer
* QT5

#### Requirements

* Installed/Working Open Fodder
* Works with any data supported by Open Fodder
